# 🏦 Banking Credit Risk Analysis

A comprehensive data analytics project that analyzes customer credit profiles to identify credit risk, evaluate loan eligibility, and support data-driven lending decisions. The project leverages SQL, Python, Power BI, and Excel to perform exploratory data analysis, develop interactive dashboards, and generate business insights for financial institutions.

---

## 📖 Overview

Credit risk assessment is a critical process in the banking industry. Financial institutions need to evaluate customer profiles before approving loans to minimize the risk of defaults.

This project demonstrates an end-to-end data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based analysis, KPI reporting, interactive dashboard development, and business recommendations to support credit risk management.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/apurva22-chavan/Banking-Credit-Risk-Analysis.git
```

Navigate to the project directory:

```bash
cd Banking-Credit-Risk-Analysis
```

Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Run the notebook to reproduce the analysis.
